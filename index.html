Om Åse

David Åse (linkedin) er utdannet Sivilingeniør i datateknologi ved NTNU Gløshaugen og en aktiv open-source entusiast. Han står bak en rekke populære open-source prosjekter:

  • Javalin - Et enkelt og moderne webrammeverk for Java og Kotlin
  • j2html - Typesafe HTML generering i Java
  • GitHub profile summary - Visualisering av GitHub profiler
  • Bubbly - Animerte bakgrunner på under 1kb

Han arbeider for tiden i startupselskapet Working Group Two, hvor han trives godt.
